## Nix package manager

<div class="my-md-content">

O'rnatish va asosiy buyruqlar.

```bash
# macOS/Linux (ruxsat kerak bo'lishi mumkin)
sh <(curl -L https://nixos.org/nix/install) --daemon

# versiyani tekshirish
nix --version
```
